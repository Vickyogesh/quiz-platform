Quiz Web Service
================

|quiz| provides driving quizzes and exams to assist clients of driving schools
in learning and preparing to pass real exam. The service provides per student
statistics and aggregated school statistics. There are several driving
categories supported each of them with separate clients statistics.

This document describes project structure, features and technical details
of the |quiz|.
